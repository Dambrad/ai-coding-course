If you've never coded before and want to start RIGHT NOW:

### Step 1: Open Terminal (2 minutes)
1. Press `Cmd + Space`
2. Type "Terminal"
3. Press Enter

You'll see a window with text. Don't worry, you got this!

### Step 2: Install Homebrew (5 minutes)
Copy this entire line and paste it into Terminal, then press Enter:

```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

Enter your Mac password when asked (you won't see it as you type - that's normal!).

Wait for it to finish (takes 5-10 minutes). When you see your username again, it's done!

### Step 3: Install Python (3 minutes)
Type or paste this:

```bash
brew install python@3.11
```

Press Enter. Wait for it to finish.

### Step 4: Check It Worked (1 minute)
Type:

```bash
python3 --version
```

If you see "Python 3.11.x" - YOU DID IT! ğŸ‰
